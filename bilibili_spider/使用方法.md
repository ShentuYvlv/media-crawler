## 4. 可选参数

- `--num`
  - 要抓取的结果数量
- `--page-size`
  - 每页请求数量，默认 `20`
- `--order`
  - 排序方式，可选：
  - `totalrank`
  - `click`
  - `pubdate`
  - `dm`
  - `stow`
  - `scores`
- `--excel-name`
  - 导出的 Excel 文件名，不需要写 `.xlsx`
- `--cookie-path`
  - 自定义 `cookie.json` 路径

## 5. 自定义 cookie.json 路径


python "/Users/zed/all code/D 互影/media-crawler/bilibili_spider/search_cli.py" "盗亦有道" --num 20 --cookie-path "/你的/cookie.json" --excel-name bili_custom_cookie

## 8. 出错时先检查

如果运行失败，优先检查这几项：

- `venv` 是否已激活
- `cookie.json` 是否存在且未过期
- 是否已经安装依赖：

pip install bilibili-api-python aiohttp
