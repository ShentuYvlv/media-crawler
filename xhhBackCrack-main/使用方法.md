# 使用方法

## 运行前提

- 需要本机有 `python3`
- 需要本机有 `node`
- 需要 [JS/local_auth.js](/Users/zed/all%20code/D%20互影/media-crawler/xhhBackCrack-main/JS/local_auth.js) 里的登录态可用

## 配置文件

项目根目录的 [ .env ](/Users/zed/all%20code/D%20互影/media-crawler/xhhBackCrack-main/.env) 用来控制默认值：

env
XHH_SEARCH_LIMIT=10
XHH_COMMENT_LIMIT=100


含义：

- `XHH_SEARCH_LIMIT`：默认搜索抓多少条帖子
- `XHH_COMMENT_LIMIT`：每个帖子最多写多少条评论到 Excel

优先级：

1. 命令行参数
2. `.env`
3. 代码默认值

## 最常用命令

进入目录：


cd "/Users/zed/all code/D 互影/media-crawler/xhhBackCrack-main"


默认搜索：


python3 xhh_api.py --search "暗影成双"


说明：

- 搜索数量默认取 `.env` 的 `XHH_SEARCH_LIMIT`
- 每帖评论数量默认取 `.env` 的 `XHH_COMMENT_LIMIT`

指定搜索数量：


python3 xhh_api.py --search "暗影成双" limit=20


指定每帖评论数量：


python3 xhh_api.py --search "暗影成双" comment_limit=30


同时指定搜索数量和评论数量：


python3 xhh_api.py --search "暗影成双" limit=20 comment_limit=30
